## Database Structure (Firestore)

### 1. Root Collection: `users`
- Each document ID is the Firebase Auth User UID.
- Stores basic account information and onboarding status.

### 2. Sub-Collection: `users/{uid}/sleepLogs`
- Each document ID is the date of the log (e.g., `2024-04-09`).
- Records sleep metrics for a specific day.

### 3. Sub-Collection: `users/{uid}/habitLogs`
- Each document ID is the date of the log (e.g., `2024-04-09`).
- Records various health habits and stress levels.

### 4. Sub-Collection: `users/{uid}/user_health_details`
- Contains a single document named `main`.
- Stores static health profile information used for disease risk calculations.
