## Component Descriptions

### 1. Presentation Layer (Vite + React)
- **Vite**: Modern frontend build tool ensuring fast development and optimized production bundles.
- **Tailwind CSS**: Utility-first CSS framework for a consistent, premium design system.
- **Recharts & Framer Motion**: Utilized for dynamic data visualization and smooth UI transitions to enhance user engagement.

### 2. Analytical Core (Prediction Engine)
- **Local Processing**: All disease risk calculations are performed client-side in `lib/predictions.ts`. This ensures zero-latency analysis and preserves user privacy.
- **Algorithm**: Processes weighted sleep logs, lifestyle habits, and physiological profiles (BMI, family history) to generate a dynamic "Health Score".

### 3. Data & Identity (Firebase)
- **Authentication**: Managed via Firebase SDK, supporting persistent secure sessions.
- **Cloud Firestore**: Stores all time-series health data (sleep/habits) and user profile information in a hierarchical NoSQL structure.
- **React Query**: Acts as the middleware between Firestore and the UI, managing caching and offline data optimistic updates.

### 4. Cross-Platform Strategy
- **Capacitor**: Enables the same React codebase to run as a native Android application.
- **GitHub Pages**: (Planned) Serves the web-accessible version of the application with automated deployment via GitHub Actions.
